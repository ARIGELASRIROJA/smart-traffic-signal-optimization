# Smart Traffic Signal Optimization System

![Traffic Signal](docs/diagrams/traffic_signal_example.png)

## 📝 Project Overview
Fixed-time traffic signals often cause congestion, long waiting times, and fuel wastage in urban areas.  
This project aims to **optimize traffic signal timings** using real-time traffic data to reduce congestion, improve traffic flow, and save fuel.

---

## 🚀 Features
- Real-time **vehicle detection** using YOLO and OpenCV
- **Dynamic traffic signal control** based on traffic density
- **Graphical visualization** of vehicle count and signal timings
- Supports **video input** or live camera feed
- Generates **simulation results and analytics**

---

## 🛠 Technology Stack
- **Python 3.10+**
- **OpenCV** – Video processing & image analysis
- **YOLO (Ultralytics)** – Object detection
- **Streamlit** – Interactive web interface
- **NumPy & Pandas** – Data processing
- **Matplotlib / Seaborn** – Visualization
